课程概览

学习目标

从最基础的串口输出和 LED 控制开始，逐步掌握 GPIO、ADC、PWM、WiFi 和舵机等常见嵌入式基础能力。

材料组织

当前文件夹已经将每个实验拆成独立子目录，避免把所有示例混在一个文件里，更适合课程提交和后续复习。

页面作用

本页相当于嵌入式入门板块的 HTML 入口页，便于集中展示实验目录、代码文件和文字学习记录。

模块安排

E1 HelloWorld

完成开发环境联通与串口输出，是整个入门实验的起点。

E2 数字输出闪灯

通过 LED 闪烁理解最基础的 GPIO 输出控制。

E3 数字输入按键

学习读取按键输入，并根据输入状态控制输出。

E4 模拟输出呼吸灯

通过 PWM 实现 LED 亮度渐变，建立模拟输出概念。

E5 模拟输入光敏

读取光敏电阻的模拟值，体验传感器输入采样。

E6 WiFi

完成 ESP32 联网初始化，为后续远程控制做准备。

E7 控制舵机

理解舵机角度控制与 PWM 驱动关系。

E8 WiFi 舵机

把联网能力和执行器控制结合起来，形成综合案例。

学习总结

知识线索
